Name

 simplemail.rb - Provides a simple interface to popen'd sendmail.

 $Id: simplemail_rb.html,v 1.4 2003/06/07 12:28:24 nigelb Exp $

Synopsis

      mail = SimpleMail.new
      mail.to = "Colin <colin@rubycookbook.org>"
      mail.from = "colin@rubycookbook.org"
      mail.sender = "bounce@rubycookbook.org"
      mail.subject = "This is a test"
      mail.text = "This is some text in the text."
      mail.html = "<b>big html stuff</b>"
      mail.headers["X-Foobar"] = "Baz"
      mail.attachementst << "/path/to/attachement/file.ext"
      mail.send

Description

SimpleMail is a simple wrapper class which popen’s sendmail(1) to send email. It supports attachements and mutlipart/mixed messages with text and html content. This makes it ideal for mailing list type applications
