from  [RW] 

mail.from = "My Name <myname@my.domain.com>" If the mail address is supplied in this format, sendmail is invoked with -F "My Name" mail.from= "myname@my.domain.com" Sets the From header.

headers  [RW] 

mail.headers["header"] = "header" sets the required header to the value supplied. Use for such purposes as mail.headers["Mail-list"} = "My mailing list". Optional.

html  [RW] 

mail.html = "html text" sets the text/html part of the e-mail message. Do not set to send a simple text only email

sender  [RW] 

mail.sender = "My Name <myname@my.domain.com>" mail.sender = "myname@my.domain.com" Sets the Return-to header and the From header if mail.from not set.

subject  [RW] 

mail.subject = "Mail message subject". Self explanitory. Optional.

text  [RW] 

mail.text = "text" sets the text/plain part of the e-mail message. One of html or text must be set.

Public Class methods
new(sendmail_command = DEFAULT_SENDMAIL_COMMAND)

sendmail_command is a string with a relative or absolute path to sendmail (or something equivalent). DEFAULT_SENDMAIL_COMMAND is "/usr/sbin/sendmail -t". Should also work with courier & qmail substitutes.

Public Instance methods
send()

High-level interface to send mail. Transmits the mail using the previously supplied body, to, from, etc.

We send one of:

  • text only
  • text and html
  • text, html and attachments
  • text and attachments
